# debajyotis.github.io

Personal research portfolio — [debajyotis.github.io](https://debajyotis.github.io)

Built with [al-folio](https://github.com/alshedivat/al-folio).

## Local development

Requires Docker.

```bash
docker compose up
```

Site available at `http://localhost:8080`. Live-reloads on file changes.

## Structure

| Path | Purpose |
|------|---------|
| `_pages/about.md` | Homepage / bio |
| `_bibliography/papers.bib` | Publications (rendered on `/publications`) |
| `_projects/` | Research project pages |
| `assets/json/resume.json` | CV data (rendered on `/cv`) |
| `_config.yml` | Site-wide settings |
